📦 브라질 이커머스(Olist) 배송 지연 리스크 및 판매자 분석

📌 프로젝트 개요
브라질 최대 이커머스 Olist 데이터셋을 활용하여 '배송 지연'의 근본 원인을 판매자(Seller)와 택배사(Carrier)로 분리합니다.
이를 통해 지연 시간을 줄이고 우수 판매자를 확보하기 위한 데이터 기반 액션 아이템을 도출합니다.

🎯 핵심 문제 정의
지연 매출 증가: 성장하는 전체 매출 이면에 숨겨진 '배송 지연 주문'의 비중 추적 필요.
책임 소재 불분명: 지연 원인이 '판매자의 늦은 출고'인지 '물류 인프라(택배사)'인지 식별 불가.
판매자 관리 미비: 매출은 높지만 상습적으로 지연을 발생시키는 '블랙 판매자' 제재 기준 부재.

📊 주요 분석 내용 (Core SQL Analysis)
RFM 기반 요주의 판매자 도출 (01-1)
매출(M)은 최상위권이나 배송 지연이 심각한 판매자를 식별하여 리스크 타겟팅.
지연 주체 분리 및 지역 분석 (01-2)
배송 기한과 실제 인계일을 비교해 '판매자 지연'과 '택배사 지연'을 분리하고 도시별로 매핑.
상품 특성(무게/부피)과 지연의 상관관계 (01-3)
판매자 책임 지연 비중이 높은 그룹일수록 상품의 무게/부피가 어떻게 달라지는지 분석.
판매자 라이프사이클 & 코호트 분석 (01-4, 01-5)
가입 첫 달의 배송 퍼포먼스(우수 vs 심각)가 판매자의 장기 생존율(유지율)에 미치는 영향 추적.

🛠 Tech Stack
Database: MySQL

💡 도출된 비즈니스 액션 (Action Items)
판매자 페널티 정책: 매출 기여도가 높아도 출고 지연율이 높은 판매자는 노출 순위 하향.
풀필먼트 우선 도입: 판매자 지연 비중이 높은 고중량/대부피 상품군 대상 플랫폼 자체 물류 지원.
신규 판매자 인큐베이팅: 가입 첫 달 배송 퍼포먼스가 장기 활동 여부를 결정하므로, 초기 모니터링 강화.
